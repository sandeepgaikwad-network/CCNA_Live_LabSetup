## 🧩 Key Concepts Demonstrated

- **VLAN Configuration** and Trunking  
- **Router‑on‑a‑Stick** setup  
- **DHCP Server** on routers  
- **Routing Protocols** – Static, RIP, EIGRP, OSPF  
- **Access Control Lists (ACLs)** – Standard and Extended  
- **NAT and PAT** implementation 

---

## 💡 How to Use

- Open each lab manual for configuration commands.  
- Watch the GIFs to understand packet movement and ACL behavior.  
- Replicate the setup in **Cisco Packet Tracer** or **GNS3** for practice.
